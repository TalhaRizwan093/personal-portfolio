## Backend Architecture Projects

### 1. Spring Security OAuth2 + JWT Starter
**Repository:** https://github.com/TalhaRizwan093/Springboot-Security-Starter-OAuth-JWT

**Technologies:** Java, Spring Boot 3, Spring Security 6, OAuth2, JWT, React, TypeScript, MySQL

**Description:** Production-ready authentication and authorization system featuring OAuth2 integration with Google and GitHub, JWT token management, and a complete React frontend. This enterprise-grade starter project implements security best practices and can be directly used as a foundation for new applications.

**Key Features:**
- OAuth2 authentication with multiple providers
- JWT token-based authorization
- Username/password authentication flow
- Comprehensive exception handling
- React frontend with modern UI
- MySQL database integration
- Industry best practices implementation
